Enable a customizable maintenace mode for your wordpress site!

== Description ==

Activating this plugin allows you to enable a customizable Maintenance Mode page, preventing 
users to navigate your under-construction wordpress based web-site.
With NS Maintenance Mode you can customize the page tha users will see visiting your site. 
By accessing the admin backend settings page you can choose the background image to use and the text to display while the plugin is activated.
This plugin also provides an integrated form to permit users to subscribe the website in order to receive information about it. You will be
notified with a custom email about new subscriptions.
A social section is included to grant the possibility to link your site to the most famous socials.
This plugin will also add a subscriber custom post type, with the 'export' useful functionality that will permit to obtain a .csv file of all subscribed users.

The customization is fast and easy.
You will be able to edit:
- Background image
- Social's links
- Text colors, button colors backgounds and border inputs colors
- Title, subtitle and footer text 
- Font of the page



*We test our plugins with many other. If you find incompatibility with one specific plugin, please inform us at <a href="mailto:report@nsthemes.com">report@nsthemes.com</a>.*


== Installation ==

1. Upload the entire 'ns-maintenance-mode.zip' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Maintenance Mode' and start to customize it.


== Frequently Asked Questions ==

= Code is required to use this plugin? =

No code require to use this plugin.

= Can a user view the content of the site while this plugin is active? =

No. Only administrators can view and navigate the site while this plugin is active.
